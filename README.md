# Mentalidad de Crecimiento y Comunicación en Entornos Digitales

## Blog Técnico

Este repositorio contiene la actividad práctica correspondiente al módulo **Mentalidad de Crecimiento y Comunicación en Entornos Digitales** de la diplomatura.

El objetivo de este proyecto es documentar un desafío técnico real utilizando un enfoque de **post-mortem constructivo**, aplicar buenas prácticas de documentación, evidenciar el uso de control de versiones mediante Git y GitHub y reflexionar sobre la importancia del feedback radicalmente sincero durante el proceso de aprendizaje.

## Contenido

- Artículo técnico: `index.md`
- Evidencia de control de versiones mediante Git
- Publicación mediante GitHub Pages

- GitHub
- GitHub Pages
